# Quiz App 
## Language used
- HTML
- Tailwind CSS
- Javascript
## Description
- A simple Quiz app 
- if the selected answer is wrong then the correct answer appears and lock the answer options
- It shows score after all the question is attempted
- It has finish quiz and play again options also
